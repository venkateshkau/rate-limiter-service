## Why Undertow
- To gain deeper understanding of server side things.
- Wildfly and Jboss were using Undertow
- Control + minimal abstraction + explicit threading model
## Why no Spring Boot
- Provides opinionated framework
- Highly layered and annotated
- Requires fundamental understanding of server side in order to troubleshoot.
## Why Maven
- Ubiquitous
- plugin ecosystem
- easy for reviewers

## Why Java 21
- LTS
- virtual threads
