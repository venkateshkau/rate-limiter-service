# rate-limiter-service

## Description

This service is consumed by the microservice in order to control the amount of user requests. This rate limiter service helps to control the income requests during particular time window, configured easily by consumers and supports various different algorithms.


## Algorithms

Here are the algorithms that are supported by this service,
 - Token Bucket
 - Sliding Window
 - Fixed Window

 
## Supports

This rate limiter service supports the following 
 - API Keys
 - Per-user limits
 - Bursty traffic
 - Distributed deployments
